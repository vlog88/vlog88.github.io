一、为 Mac 追加【ls -l】命令

		编辑文件：
		vim ~/.bash_profile
		
		输入命令别名：
		alias ll='ls -alF'
		alias la='ls -A'
		alias l='ls -CF
		
		保存：
		:wq
		
		立即生效：
		source ~/.bash_profile

二、为 Mac 安装 nginx

		安装nginx
		brew install nginx
		
		查看安装信息
		brew info nginx
		
		nginx常用命令
		brew services start nginx
		brew services restart nginx
		brew services stop nginx
		
		服务地址
		http://127.0.0.1:8080/
		
		创建软连接：
		ln -s  /Users/cyril/Downloads/processing /usr/local/var/www/p
